# 🛒 Shopping Bill Management System Using Python

A Python-based console application that simulates a real-world shopping billing system. The program calculates the total purchase amount, applies membership and festival discounts, computes GST, and generates a detailed customer bill.

---

## 📌 Project Description

The **Shopping Bill Management System** is designed to automate the billing process for retail purchases. It accepts customer details, calculates discounts based on membership type, applies an additional festival discount for eligible purchases, calculates GST, and displays the final payable amount.

This project is ideal for beginners who want to practice Python fundamentals such as user input, conditional statements, arithmetic operations, and formatted output.

---

✨ Features

- Accepts customer details.
- Calculates total purchase amount.
- Applies membership-based discounts.
- Applies festival discount for eligible purchases.
- Calculates 18% GST after discounts.
- Generates a detailed shopping bill.
- Clean and easy-to-read console output.
- Demonstrates real-world billing logic.

---

🛠 Technologies Used

- Python 3
- Variables
- User Input
- Arithmetic Operators
- Conditional Statements (`if`, `elif`, `else`)
- Console Output Formatting

---

📊 Business Logic

| Condition | Discount |
|-----------|----------|
| Gold Membership | 20% |
| Silver Membership | 10% |
| Regular Membership | 0% |
| Purchase Above ₹20,000 | Additional 5% Festival Discount |
| GST | 18% on Discounted Amount |
